# 画像・ビジュアルルール

## OGP画像プロンプト
- 設計書は `zenn-repo/images/{slug}/ogp-design.md` に保存する
- プロンプトは英語で作成する（画像生成AIの再現度向上のため）
- 実際の画像生成は人間がGeminiのNanobananaで行う
- OGP推奨サイズ: 1200×630px

## プロンプトに含める要素
- 画像の目的（アイキャッチ / 図解 / 概念図）
- 構図（レイアウトの詳細指示）
- スタイル（フラットデザイン / イラスト / ダイアグラム等）
- カラーパレット
- 含める要素・除外する要素

## Mermaid図解
- 技術的な概念やフローの図解にはMermaid記法を使用する
- 記事内に直接埋め込む
- フローチャート、シーケンス図、クラス図等を適切に選択する
- `graph TD` / `graph LR` 等の安定記法を使用する（`block-beta` 等の新しい記法はZennで非対応の可能性あり）

## 画像の記事への設置
1. ユーザーが画像を生成・配置したことを確認する
2. `zenn-repo/images/{slug}/` に画像ファイルが存在することを確認する
3. 記事のfrontmatter直後に `![](/images/{slug}/ogp-design.png)` を追記する
